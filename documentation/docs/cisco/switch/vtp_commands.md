# Cisco - VTP

## VTP
### Afficher le statut du VTP
```
show vtp status
```

### Activer le VTP
- Commande :
```
vtp domain ...
```
- Exemple :
```
vtp domain eLearnCisco
```

### DÃ©finir le mode de VTP (Serveur, Client ou Transparent)
```
vtp mode ...
```
- Choix entre :
    - **server**
    - **client**
    - **transparent**

## VTP pruning
### Activer le VTP pruning
```
vtp pruning
```
